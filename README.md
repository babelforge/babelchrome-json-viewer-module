# JSON Viewer Module

`babelforge.json-viewer` is an installable BabelChrome viewer module for JSON documents.

It renders local and remote JSON sources through Symfony, `andypf/json-viewer`, Twig templates, AssetMapper assets, and `babelforge/babel-chrome-viewer-kit`.
