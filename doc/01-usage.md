# Usage

Navigation: [README](README.md) | [Next: Development](02-development.md)

The module handles `json` sources.

Stable routes include:

```text
babelchrome://json/file/<encoded-path>
babelchrome://json/url/<encoded-url>
```

The viewer displays a JSON badge in the BabelChrome address field and exposes the shared `Open with` header control for local files.
